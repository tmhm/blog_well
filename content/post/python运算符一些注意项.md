###  '/'浮点除，和'//'整除

单个'/'是浮点除，两个除号'//'是整除

整除也适用于浮点数。但是，用整除计算浮点除的结果只是在整除的结果上*浮点化*，比如3.6//2.1,结果是1.0。

例子：

```
a = 5
for i in [1.0,1.5,2,3,4,5,6]:
        b= 5//i
        c = 5/i
        print a, b,c
print "3.6/2.1 =",3.6/2.1
print "3.6//2.1 =",3.6//2.1
```

out：

```
5 5.0 5.0
5 3.0 3.33333333333
5 2 2
5 1 1
5 1 1
5 1 1
5 0 0
3.6/2.1 = 1.71428571429
3.6//2.1 = 1.0
```
